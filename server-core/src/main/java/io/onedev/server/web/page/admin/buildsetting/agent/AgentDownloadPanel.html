Download & Install Agent
  • Make sure Server URL in system setting is specified correctly
  • On remote machine (Windows/Mac/Linux/FreeBSD) where you want to install agent:
    1. Make sure JDK 1.8 or higher is installed
    2. If you plan to execute jobs inside containers on this agent, make sure that::
      • Docker engine is installed and docker command line is available in system path
      • Make sure current user has permission to run docker containers
    3. Make sure git version 2.11.1 or higher is installed and available in system path
    4. Download agent package. A new agent token will be included in the package
    5. Extract the package into a folder
    6. From extracted folder, run bin\agent.bat console as administrator on Windows or bin/agent.sh console on other OS
  • Check agent management for more details, such as running as service etc.
